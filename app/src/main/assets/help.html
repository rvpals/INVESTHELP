Dashboard

Your portfolio overview at a glance.

  • Market Indices — horizontal row of cards showing NASDAQ, S&P 500, Dow, Gold and more. Tap a card to open Yahoo Finance for that index. Customize which indices appear in Settings > Preferences.
  • Daily Glance — shows overall daily change for Stocks and ETFs, plus the top 5 gainers and top 5 losers today. Toggle "By Per Share" to switch between total and per-share view. Tap a row to open that item.
  • Positions Pie Chart — all tickers by allocation. Legend shows top 20 with a "More" button. Tap a ticker in the legend to navigate to item detail.
  • Position Details — scrollable table with current price, total cost, total value, and gain/loss per ticker.
  • All dashboard cards are collapsible. Tap the pin icon to keep a card expanded by default.

Items (Positions)

Manage every investment you hold.

  • Each item is identified by its ticker symbol (one record per ticker, not per account).
  • Supports types: Stock, ETF, Bond, Mutual Fund, Crypto, Other.
  • Sort by Ticker, Total Value, or Current Price via dropdown.
  • Tap Refresh All in the toolbar to fetch the latest prices and company names from Yahoo Finance.
  • Tap an item to see a detail screen with stats, daily high/low, transactions, and analysis.
  • Item detail shows Analysis Info (historical highs and lows) and a link to Yahoo Finance.
  • New tickers are auto-created when you record a transaction for a ticker that doesn't exist yet.

Transactions

Record every buy and sell.

  • Fields: ticker, shares, price per share, date (optional time), account, total amount (verification), and note.
  • Analyze Price — button next to the Price field. Opens a screen showing current price, your transaction avg/max/min, and historic high/low across multiple time ranges. Tap a price to copy it back to the form.
  • View — button next to Ticker opens item detail without losing form state.
  • Simulate — calculates days since transaction date and opens simulation with that custom range.
  • Account is auto-selected for new transactions.
  • Multi-select — long-press a transaction to enter selection mode. Use Select All and bulk Delete. Respects the "Warn before delete" setting.
  • Each transaction card shows gain/loss and days since purchase.

Bank Transfers

Track fund transfers into investment accounts.

  • Records date, amount, destination account, and an optional note.
  • Summary at the top shows total amount transferred per account.

Simulation

Project future value based on historical data.

  • Select a ticker and number of shares.
  • Time ranges: 1W, 2W, 1M, 3M, 6M, 1Y, 2Y, 5Y, 10Y, MAX (grouped into Week / Month / Year rows).
  • Tap a point on the chart to see a tooltip with price and date.
  • Custom day ranges when launched from the Transaction form's "Simulate" button.

Accounts

Organize your investment accounts.

  • Create accounts for different brokerages, IRAs, 401k plans, etc.
  • Transactions and bank transfers reference accounts.
  • Deleting an account cascades — it also removes all associated transactions and bank transfers.

Performance

Track portfolio value over time.

  • Add records manually or tap "Pull from App" to auto-fill the current total value.
  • Optional note per record (edit via pencil icon).
  • Multi-account chart — select accounts via FilterChip toggles. Each gets a distinct color.
  • Pinch to zoom (1x–5x), two-finger pan, tap for tooltip, double-tap to reset.
  • Unique constraint: one record per account per date.

Watch List

Track tickers you're interested in.

  • Create multiple named watch lists, switch via FilterChip selector.
  • Add a ticker with shares count and price-when-added.
  • Tap "Fetch" to pull the current market price from Yahoo Finance.
  • Table shows: ticker, shares, current price, added price, $ change, % change, date added.
  • Rename or delete watch lists from the top controls.

Settings

Configure app behavior and manage data.

  • Preferences — toggle which market indices appear on the Dashboard (8 available). Toggle "Warn before delete" (on by default).
  • Backup & Restore — back up or restore the entire database. Choose a backup folder that persists across sessions.
  • CSV Import — import Position Details, Transaction Records, or Performance Records from CSV files. Supports column mapping with brokerage aliases (e.g., Symbol → ticker, Description → name). Includes a 3-row preview and account selector.
  • Mappings are saved per import type so you don't have to remap each time.

SQL Explorer

Power-user tool for direct database access.

  • Run any SQL query — SELECT, INSERT, UPDATE, DELETE, PRAGMA, and more.
  • Table browser — lists all tables with expandable column details (name, type, PK/NN).
  • Tap "Open" on a table to run SELECT * FROM table.
  • Result grid with horizontal and vertical gridlines. Tap a row to view full field values.
  • CSV Export — share query results as a CSV file.

Top Bar & Portfolio Card

  • The portfolio card at the top always shows Total Portfolio Value with daily change in parentheses, color-coded green/red.
  • Second row shows daily % change and all-time % change.
  • Tap the card to refresh all prices from Yahoo Finance and navigate to Dashboard.
  • A progress bar appears while prices are refreshing.

Tips & Shortcuts

  • Tap the portfolio card at the top to refresh all prices at once.
  • Company logos load automatically from the web.
  • New tickers are auto-created when you add a transaction for an unknown ticker.
  • Pin dashboard cards to keep them expanded by default.
  • Use Analyze Price to see historical price ranges before buying.
  • Long-press a transaction to start multi-select mode.
  • In Performance charts, double-tap to reset zoom.
  • CSV import auto-detects common brokerage column names (Symbol, Description, Price, Quantity, etc.).
  • The app log (About > Show Log) captures price fetch results and errors for debugging.
